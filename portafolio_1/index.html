Primera practica

555

Integrado 555


RESUMEN:

La práctica se centró en la implementación de un circuito de reloj con el circuito integrado 555, controlado por un potenciómetro para ajustar la frecuencia de parpadeo de un LED. Se diseñó y simuló en diversos medios, lo que facilitó la visualización y anticipación de problemas antes del ensamblaje físico. Además de alcanzar el objetivo técnico, la actividad permitió a los participantes familiarizarse con diversos componentes electrónicos y mejorar sus habilidades en el armado

INTRODUCCION:

Las señales de reloj son componentes esenciales en la lógica digital, actuando como el pulso que sincroniza los sistemas. Estas señales se caracterizan por sus oscilaciones periódicas entre estados alto y bajo, creando un ritmo constante que coordina el funcionamiento de los componentes digitales. En nuestra práctica de laboratorio, aplicamos este concepto fundamental para diseñar y construir un circuito que genera una señal de reloj visible. El objetivo principal fue controlar el parpadeo de un LED utilizando esta señal, proporcionando una demostración armada de los principios de sincronización digital.

MATERIALES:
  1. Protoboard.
  2. Resistencias 1kΩ
  3. Circuito Integrado 555.
  4. Batería 9V.
  5. Potenciómetro 10kΩ.
  6. Resistencia 220Ω.
  7. Transistor NPN.
  8. LED.
  9. Capacitor 47uF.
  10. Cables.
DESAROLLO:

El desarrollo del circuito se realizó en varias etapas, comenzando con su diseño como diagrama, lo que permitió familiarizarse con la interfaz y observar el comportamiento esperado. Luego, el diseño se simuló en Tinkercad, facilitando la visualización del montaje en un entorno virtual que nos permitio anticipar problemas antes de la construcción física. Finalmente, se ensambló el circuito con componentes reales. En este proceso destacó la importancia de la planificación a la hora del armado para evitar problemas.

Segunda practica

Pagina Web Sencilla

Pagina web


RESUMEN:

La práctica consistió en un primer acercamiento básico a diversas funciones de HTML, donde los participantes aprendieron a estructurar páginas web simples. A lo largo de la actividad, se exploraron elementos fundamentales como la inserción de imágenes, la creación de hipervínculos tanto internos como externos, y el uso de etiquetas básicas para formatear contenido. Posteriormente, los archivos HTML creados fueron subidos a GitHub, proporcionando una introducción al control de versiones y al uso de plataformas de alojamiento de código. Esta práctica nos permitió entender las bases del desarrollo web.

INTRODUCCION:

Una página web es un documento digital creado para ser visualizado en navegadores web, formando parte de la World Wide Web. El HTML, conocido como Lenguaje de Marcado de Hipertexto, es el estándar utilizado para estructurar y presentar contenido en la web. Esta práctica se centra en la creación de una página web básica, con el objetivo de comprender los principios fundamentales del desarrollo web

Tercera practica

Arduino 1 / 2 / 3

Práctica 0: Arduino Básico, Salidas Digitales


RESUMEN:

Las salidas digitales permiten enviar señales a un circuito de control para la diagnosticación de una función que el usuario desee. En la primera práctica aprendimos a programar con Arduino y crear modelos simulados, utilizando las funciones básicas. Generamos diferentes circuitos, utilizando las salidas digitales para encender diodos y apagarlos en un determinado momento. Además, utilizamos dichas salidas para controlar un display que muestre un función loop de un contador del 0 al 2.
